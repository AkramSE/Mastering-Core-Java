# ☕ Mastering Core Java: From Basics to Advanced

Welcome to my **Core Java** learning repository! This project serves as a comprehensive, well-structured roadmap of my journey through Java programming. It covers everything from fundamental syntax to advanced concepts like Multithreading, the Collections Framework, and File Handling.

This repository is specifically designed to build a rock-solid foundation for **Enterprise Backend Development** and **Spring Boot**.

---

## 📂 Repository Structure

The code is organized into clean, industry-standard modules. Here is what you will find inside:

### 🔹 1. Object-Oriented Programming (OOP)
* `05-OOPs/` - Classes, Objects, Constructors, and 'this' keyword.
* `06-Inheritance/` - Single, Multilevel, and Hierarchical Inheritance with 'super'.
* `07-AbstractClasses/` - Abstract classes and methods.
* `08-Interfaces/` - Multiple inheritance, Default methods, and Polymorphism.
* `09-Polymorphism/` - Compile-time and Run-time Polymorphism (Dynamic Method Dispatch).
* `10-AccessModifiers/` - Public, Private, Protected, and Default scopes.

### 🔹 2. Core Mechanisms
* `11-Multithreading/` - Thread class, Runnable interface, Priorities, sleep(), and join().
* `12-ExceptionHandling/` - Try-Catch, nested blocks, custom exceptions, and the 'finally' block.

### 🔹 3. Data Structures & Algorithms (Collections Framework)
* `13-CollectionsFramework/`
  * `ArrayList/` - Dynamic arrays and List operations.
  * `LinkedList/` - Doubly linked list operations (O(1) insertions/deletions).
  * `ArrayDeque/` - Double-ended queue operations.
  * `HashSet/` - Unique elements, hashing, and Load Factors.

### 🔹 4. Advanced Java & Utilities
* `14-DateAndTime/` - Legacy `java.util.Date` and modern `java.time` API.
* `15-AdvancedJava/` - Java Generics, Annotations, Lambda Expressions, and Javadocs.
* `16-FileHandling/` - CRUD operations on files (Create, Read, Write, Delete).

## 🚀 Mini Projects (CLI Applications)

To apply the theoretical concepts, I have built several interactive Command Line Interface (CLI) applications using Object-Oriented principles and Scanner inputs. You can find these in the **`MiniProjects/`** folder:

1. **📚 Advanced Library Management System (v2.0):** An interactive library app using `ArrayList` and custom `Book` objects to add, issue, and return books.
2. **🧮 Secure Custom Calculator:** A robust calculator demonstrating multiple custom exceptions (e.g., `MaxInputException`, `CannotDivideByZeroException`).
3. **🎮 Rock, Paper, Scissors (Best of 3):** A fun, loop-based game against a randomized computer opponent.
4. **🔢 Guess the Number Game:** A logic-building game tracking user attempts with hints.
