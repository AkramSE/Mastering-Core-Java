## 🌈 What's Inside This Repository?

This project serves as a comprehensive, well-structured roadmap of my journey through Java programming. The code is completely modular, heavily commented, and follows industry standards.

### 🧱 1. Object-Oriented Programming (OOP)
| Folder | Concepts Covered |
| :--- | :--- |
| 🔵 **`05-OOPs`** | Classes, Objects, Constructors, and the `this` keyword. |
| 🟢 **`06-Inheritance`** | Single, Multilevel, Hierarchical Inheritance, and `super`. |
| 🟡 **`07-AbstractClasses`** | Abstract classes and abstract methods. |
| 🟠 **`08-Interfaces`** | Multiple inheritance, Default methods, and Interfaces. |
| 🔴 **`09-Polymorphism`** | Compile-time and Run-time Polymorphism (Method Dispatch). |
| 🟣 **`10-AccessModifiers`** | Public, Private, Protected, and Default scopes. |

### ⚡ 2. Core Java Mechanisms
* 🚀 **`11-Multithreading/`** — `Thread` class, `Runnable` interface, Priorities, `sleep()`, and `join()`.
* 🛡️ **`12-ExceptionHandling/`** — Try-Catch, Nested blocks, Custom exceptions, and `finally`.

### 🗄️ 3. Data Structures & Algorithms (Collections)
* 📦 **`13-CollectionsFramework/`**
  * 🔹 **`ArrayList/`** - Dynamic arrays and List operations.
  * 🔹 **`LinkedList/`** - Doubly linked list operations (O(1) insertions/deletions).
  * 🔹 **`ArrayDeque/`** - Double-ended queue operations.
  * 🔹 **`HashSet/`** - Unique elements, hashing, and Load Factors.

### 🛠️ 4. Advanced Java & Utilities
* ⏳ **`14-DateAndTime/`** — Legacy `java.util.Date` vs Modern `java.time` API.
* 🧠 **`15-AdvancedJava/`** — Java Generics, Annotations, Lambda Expressions, and Javadocs.
* 📁 **`16-FileHandling/`** — CRUD operations on files (Create, Read, Write, Delete).

## 🎮 Mini Projects (CLI Applications)

To apply the theoretical concepts, I have built several interactive Command Line Interface (CLI) applications using Object-Oriented principles and User Inputs (`Scanner`). You can find these in the **`MiniProjects/`** folder:

* 📚 **Advanced Library Management System (v2.0):** An interactive library app using `ArrayList` and custom `Book` objects to add, issue, and return books. Tracks available vs issued books dynamically!
* 🧮 **Secure Custom Calculator:** A robust calculator demonstrating multiple custom exceptions (e.g., `MaxInputException`, `CannotDivideByZeroException`).
* ✊ **Rock, Paper, Scissors (Best of 3):** A fun, loop-based game against a randomized computer opponent.
* 🔢 **Guess the Number Game:** A logic-building game tracking user attempts with hints.
